# Calculadora-IMC
Struct
Projeto de uma calculadora de índice de Massa corporal onde aplico conhecimentos sobre estruturas de dados heterogeneas.
Onde os dados são salvos em váriaveis heterogeneas.

<li>Compilador MinGW</li>
</ul>
